# 2026-05-20 Coding Test

## 문제 1

- 사이트: Programmers
- 문제명: 짝수와 홀수
- 난이도: Lv. 1

### 처음 접근

```text
홀수와 짝수를 구분하는 방법
수를 2로 나눈 나머지가 0이라면 짝수
```

### 처음 생각한 코드

```java
String answer = "";
answer = num % 2 == 0 ? "Even" : "Odd";
return answer;
```

### 풀이 코드

```java
return (num % 2 == 0) ? "Even" : "Odd";
```

### 배운점

- 짝수와 홀수는 `% 2` 연산으로 구분할 수 있다.
- 조건이 단순할 때는 삼항 연산자를 사용하면 코드를 줄일 수 있다.

## 문제 2

- 사이트: Programmers
- 문제명: 평균 구하기
- 난이도: Lv. 1

### 처음 접근

```text
배열을 조회하며 총합을 구하기
구한 총합을 배열의 길이로 나누기
```

### 처음 생각한 코드

```java
double answer = 0;
for(int num : arr){
  answer += (double)num;
}
answer /= arr.length;
return answer;
```

### 풀이 코드

```java
return Arrays.stream(arr).average().orElse(0);
```

### 배운점

- Arrays의 stream 문법을 활용하여 빠르게 문제를 해결할 수 있다.

### 다시 볼 부분

- stream을 사용할 때도 반환 타입을 먼저 확인

## 문제 3

- 사이트: Programmers
- 문제명: 하샤드 수
- 난이도: Lv. 1

### 처음 접근

```text
주어진 수의 각 자리수의 합을 구하기
  1. 수를 문자로 만든 후 배열로 전환하여 각 자리수를 조회
  2. % 연산자와 / 연산자를 활용하여 수를 구하기
1번 선택
```

### 처음 생각한 코드

```java
boolean answer = true;
char[] charNumbers = String.valueOf(x).toCharArray();
int sum = 0;

for(char chNum : charNumbers){
  sum += (int)(chNum - '0');
}

if(x % sum != 0)    answer = false;

return answer;
```

### 풀이 코드

```java
int sum = String.valueOf(x).chars().map(ch -> ch - '0').sum();
return x % sum == 0;
```

### 배운점

- String.valueOf()로 숫자를 문자열로 바꿀 수 있다.
- 문자열을 toCharArray()로 문자 배열로 바꾸면 각 자릿수를 조회할 수 있다.
- 숫자 문자는 '0'을 빼면 실제 정수값으로 변환할 수 있다.

### 다시 볼 부분

- chars()와 map()을 사용하면 stream 방식으로도 자릿수 합을 구할 수 있다.

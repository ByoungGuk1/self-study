## 문제 1

- 사이트: Programmers
- 문제명: JadenCase 문자열 만들기
- 난이도: Lv. 2

### 처음 접근

```text
단어의 첫 문자는 대문자, 그 외의 알파벳은 소문자, 알파벳이 아닌 글자는 그대로
첫 문자가 대문자가 아닌경우 그대로 이후엔 소문자

문장의 형태로 들어오며 " "으로 단어 구분

조건 1 - 첫 문자인지
조건 2-1 - 알파벳인지
공백 문자의 경우 flag 변경
```

### 내 풀이

```java
String answer = "";
boolean firstWord = true;

for(int i = 0; i < s.length(); i++){
  char nowWord = s.charAt(i);
  char changeWord = s.charAt(i);
  if(firstWord && nowWord != ' '){
    if(nowWord >= 'a' && nowWord <= 'z'){
      changeWord = (char)(s.charAt(i) - 'a' + 'A');
    }
    firstWord =false;
  } else {
    if(nowWord == ' '){
      firstWord = true;
    }
    if(nowWord >= 'A' && nowWord <= 'Z'){
      changeWord = (char)(s.charAt(i) - 'A' + 'a');
    }
  }
  answer += changeWord;
}
return answer;
```

### 다른 풀이

```java
String answer = "";
String[] sp = s.toLowerCase().split("");
boolean flag = true;

for(String ss : sp) {
  answer += flag ? ss.toUpperCase() : ss;

```

### 배운점

- 공백문자가 연속으로 들어오는 경우를 구분하기 위해 각 문자별로 반복하고, `flag` 설정
- 다른 풀이처럼 모든 문자를 소문자로 변경한 후 각 문자의 첫번째만 대문자로 변환하기

### 다시 볼 부분

- String에 + 를 하기보단 StringBuilder 사용해보기
- `split(" ")`을 사용하면 연속 공백, 앞뒤 공백 처리가 곤란하다.
- `Character.toUpperCase()`, `Character.toLowerCase()`를 사용해보기

## 문제 2

- 문제명: 올바른 괄호
- 난이도: Lv. 2

### 처음 접근

```text
여는 괄호가 들어가면 +1, 닫는 괄호가 들어가면 -1
순환 중 음수가 되는경우 올바르지 않은 괄호로 판단하고 반환하기
```

### 내 풀이

```java
boolean answer = true;
int log = 0;
for(int i = 0; i < s.length(); i++){
  char nowChar = s.charAt(i);
  if(nowChar == '('){
    log++;
  }
  if(nowChar == ')'){
    log--;
  }
  if(log < 0){
    answer = false;
    break;
  }
}
if(log == 0){
  answer = true;
} else{
  answer = false;
}

return answer;
```

## 문제 3

- 사이트: Programmers
- 문제명: 최솟값 만들기
- 난이도: Lv. 2

### 처음 접근

```text
int[] a, b : 길이 동일
각 배열에서 한개씩 뽑아 곱하기 -> 배열의 길이만큼 반복
두 수를 곱한 값을 누적합
이 때 결과를 최소로 하는것이 목표
각 배열을 오름차순으로 정렬
A는 0번부터, B는 뒤에서부터 조회하면서 곱하기
```

### 내 풀이

```java
import java.util.Arrays;

class Solution
{
    public int solution(int []A, int []B)
    {
        int answer = 0;
        int length = A.length;

        Arrays.sort(A);
        Arrays.sort(B);

        for(int i = 0; i < length; i++){
            answer += A[i] * B[length - i - 1];
        }

        return answer;
    }
}
```

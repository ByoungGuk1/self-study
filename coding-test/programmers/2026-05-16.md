## 문제 1

- 사이트: Programmers
- 문제명: 문자열을 정수로 바꾸기
- 난이도: Lv. 1

### 처음 접근

```text
문자열을 숫자로 바꾸는 메서드
Integer.parseInt(문자열 변수);

문자열을 배열로 바꾸는 메서드
문자열 변수.toCharArray();

1. 입력 문자열을 문자 배열로 만들고 첫번째 값 조회
1-1. 0번 인덱스가 '-' 인 경우
      1번 인덱스부터 끝까지를 연결한 문자열 number 생성
      문자열을 숫자로 변경
      0에서 해당 숫자를 뺀 값을 결과값에 저장
1-2. 0번 인덱스가 숫자인 경우 (else)
      입력된 문자열을 숫자로 변경
      해당값을 결과값에 저장
만든 결과값을 반환
```

### 처음 생각한 코드

```java
char[] charr = s.toCharArray();

if(charr[0] == '-'){
  String numberString = "";
  for(int i = 1; i < s.length(); i++){
      numberString += charr[i];
  }
  answer -= Integer.parseInt(numberString);
} else{
  answer = Integer.parseInt(s);
}
```

### 풀이 코드

```java
answer = Integer.parseInt(s);
```

### 배운점

- `Integer.parseInt()`는 양수 문자열뿐 아니라 `"-123"` 같은 음수 문자열도 바로 정수로 변환
- 부호를 직접 분리해서 처리할 필요가 없다

### 다시 볼 부분

- `Integer.parseInt(a)`는 음수 값도 된다

## 문제 2

- 사이트: Programmers
- 문제명: 약수의 합
- 난이도: Lv. 1

### 처음 접근

```
약수를 구하기
  #1
    입력값의 절반까지만 반복문 사용
    해당 인덱스로 나눈 나머지의 값이 0인 경우
    해당 인덱스와 인덱스로 나눈 값을 더함
    중복되는 계산이 있어서 실패

  #2
    입력값의 크기 만큼 반복문 사용
    해당 인덱스로 나눈 나머지의 값이 0인 경우
    answer에 저장
```

### 처음 생각한 코드

```java
for(int i = 1; i <= n; i++){
  if(n % i == 0){
    answer += i;
  }
}
```

```java

  }
```

- 해당 인덱스만 넣고 자기 자신의 값은 따로 추가

## 문제 3

- 사이트: Programmers
- 문제명: 자릿수 더하기
- 난이도: Lv. 1

### 처음 접근

```
입력 값을 문자열로 전환
각 자리수의 합을 구하기
```

### 처음 생각한 코드

```java
String numberString = ""+n;

char[] charr = numberString.toCharArray();

for(int i = 0; i < numberString.length(); i++){
  answer += (int)(charr[i] - '0');
}
```

### 풀이 코드

```java
while(true){
  answer+=n%10;
  if(n<10)
    break;
  n=n/10;
}
```

### 배운점

- 숫자의 각 자릿수는 문자열 변환 없이 `% 10`, `/ 10` 연산으로 구하기
- `n % 10`은 마지막 자릿수를 구하고, `n / 10`은 마지막 자릿수를 제거

### 다시 볼 부분

- 나머지 연산은 생각 했지만, while문과 탈출 조건을 생각 못했다. 조금 더 집중해서 풀 수 있길

## 문제 4

- 사이트: Programmers
- 문제명: 자연수 뒤집어 배열로 만들기
- 난이도: Lv. 1

### 처음 접근

```
1. 입력 정수를 문자 배열로 전환
2. 반복문을 돌면서
3. 제출할 배열에 뒤에서부터 담기
```

### 처음 생각한 코드

```java
String nString = "" + n;
int size = nString.length();
answer = new int[size];
for(int i = 0; i < size; i++){
  answer[size - i - 1] = (int)(nString.charAt(i) - '0');
}
```

### 풀이 코드

```java
  answer = new StringBuilder()
                .append(n)
                .reverse()
                .chars()
                .map(Character::getNumericValue)
                .toArray();
```

### 배운점

- 문자열의 각 문자는 charAt(`index`)으로 꺼낼 수 있다
- 스트링 빌더 클래스 공부해보기

### 다시 볼 부분

- 문자열에서 특정 인덱스 값의 문자를 가져오고 싶다면 charAt(index) 메서드 사용
- 입력값을 문자열로 만들고, 역순으로 정렬시키고, 문자로 바꾸고, 문자를 숫자로 바꾸고, 정수배열로 만들기

## 문제 5

- 사이트: Programmers
- 문제명: 정수 내림차순으로 배치하기
- 난이도: Lv. 1

### 처음 접근

```
1. 자리수만큼 반복문 돌기
2. 작은 수부터 오름차순으로 우선 정렬
  2-1. 2중 for문 사용
      가장 작은 수를 찾을 때 까지 배열 조회
      찾으면 맨 앞과 교체
3. 역순으로 조회하면서 answerString 만들기
4. 정수형으로 변환하고 제출
```

### 처음 생각한 코드

```java
String nString = "" + n;
int size = nString.length();
char[] charArr = nString.toCharArray();
String answerString = "";

for(int i = 0; i < size; i++){
  char tmp = charArr[i];
  for(int j = i; j < size; j++){
    if(tmp > charArr[j]){
      charArr[i] = charArr[j];
      charArr[j] = tmp;
      tmp = charArr[i];
    }
  }
}

for(int i = 0; i < size; i++){
  answerString += charArr[size - i - 1];
}

answer = Long.parseLong(answerString);
```

### 풀이 코드

```java
String[] list = String.valueOf(n).split("");
  Arrays.sort(list);

  StringBuilder sb = new StringBuilder();
  for (String aList : list) sb.append(aList);

  return Long.parseLong(sb.reverse().toString());
```

### 배운점

- 직접 이중 반복문으로 정렬할 수도 있지만, Arrays.sort()를 사용하면 더 간단하게 정렬 가능
- 문자열을 반복해서 붙일 때는 StringBuilder를 쓰는 방식도 있다

### 다시 볼 부분

- 직접 정렬 로직을 구현하는 방식과 Arrays.sort()를 사용하는 방식 비교하기
- 오름차순 정렬 후 reverse()로 내림차순을 만드는 패턴 기억하기
- String을 반복해서 += 하는 방식보다 StringBuilder가 더 좋다

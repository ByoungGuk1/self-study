# 2026-05-19 Coding Test

## 문제 1

- 사이트: Programmers
- 문제명: 정수 제곱근 판별
- 난이도: Lv. 1

### 처음 접근

```text
1. n이 x의 제곱인지 아닌지 판단하기
  조건 1
    -> n = x^2
    -> n == x * x
  조건 2
    -> 루트 n인 i 는 n / i 보다 클 수 없다
2.
  true -> return (x + 1) * (x + 1)
  false -> return -1

```

### 처음 생각한 코드

```java
for(long i = 1; i <= (n / i); i++){
  if(i * i == n){
    return ((i + 1) * (i + 1));
  }
}
return -1;
```

### 풀이 코드

```java
    double i = Math.sqrt(n);
    return Math.floor(i) == i ? (long) Math.pow(i + 1, 2) : -1;
```

### 배운점

- Math의 메서드 외우기
- 루트한 값을 버림 처리 했을 때, 버리기 전과 값이 동일하다면 그 수는 정수이다.

### 다시 볼 부분

- 입력된 값이 long 타입이기 때문에, 반복문을 돌릴 때 i의 타입도 long으로 두는 것이 안전하다.
- i \* i <= n 대신 i <= n / i 를 사용하면 오버플로우 상황을 피할 수 있다.
